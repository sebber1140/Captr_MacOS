# Logo Files

**ACTION REQUIRED:** Replace the placeholder logo files with the actual Captr branding:

- `captr.png` - Main application icon (PNG format)
- `captr.ico` - Windows icon format (ICO)
- `captr.svg` - Original SVG logo (if available)

The current `captr.png` and `captr.ico` files are placeholders copied from the original DuckTrack logos.

To update the logo:
1. Replace `captr.png` with your Captr logo (recommended size: 512x512 or 1024x1024 pixels)
2. Replace `captr.ico` with your Captr icon
3. Optionally add `captr.svg` for the original vector graphic
4. After replacing, rebuild the application using `python build.py`

The old DuckTrack logo files (`duck.png`, `duck.ico`) can be kept for reference or deleted.
